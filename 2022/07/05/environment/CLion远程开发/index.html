CLionиҝңзЁӢејҖеҸ‘

з®Җд»Ӣ

LinuxејҖеҸ‘зҺҜеўғе№¶дёҚеҸӢеҘҪпјҢVSCodeиҷҪз„¶жңүдәҶRemote SSHдҪҶд»Қз„¶и§үеҫ—дёҚеҸӢеҘҪпјҢCLion жҳҜдёҖж¬ҫдё“дёәејҖеҸ‘ C еҸҠ C++ жүҖи®ҫи®Ўзҡ„и·Ёе№іеҸ° IDEгҖӮ е®ғжҳҜд»Ҙ IntelliJ дёәеҹәзЎҖи®ҫи®Ўзҡ„пјҢеҢ…еҗ«дәҶи®ёеӨҡжҷәиғҪеҠҹиғҪжқҘжҸҗй«ҳејҖеҸ‘дәәе‘ҳзҡ„з”ҹдә§еҠӣпјҢд»ҺиҖҢжҸҗеҚҮе·ҘдҪңж•ҲзҺҮгҖӮ

зҺҜеўғеҮҶеӨҮ

  • жң¬жңәпјҡmacOS / Windows
  • IDE: CLion
  • дёҖеҸ°дә‘дё»жңә

жң¬ж–ҮйҮҮз”Ёзҡ„жҳҜи…ҫи®Ҝдә‘зҡ„ubuntu 20.04е’ҢmacosпјҢwindowsзҡ„и®ҫзҪ®ж–№ејҸеә”иҜҘеӨ§еҗҢе°ҸејӮгҖӮ

иҝһжҺҘеҮҶеӨҮ

йҰ–е…ҲдҪҝз”Ёз»Ҳз«ҜSSHиҝӣиЎҢиҝһжҺҘпјҢеүҚеҫҖ~/.sshзӣ®еҪ•дёӢпјҢжү§иЎҢе‘Ҫд»Ө

vim config

жҠҠдёӢйқўеҶ…е®№зІҳиҙҙиҝӣеҺ»е№¶дҝқеӯҳ

Host T					//иҝҷйҮҢзҡ„TеҸӘжҳҜдёҖдёӘеҗҚеӯ—
 HostName ****	//иҝҷйҮҢжҳҜip
 User	ubuntu	//иҝҷйҮҢжҳҜз”ЁжҲ·еҗҚ
 Port 	22			//з«ҜеҸЈеҸ·
 IdentityFile ~/.ssh/tencent_key	//еӯҳж”ҫз§Ғй’Ҙ
 ForwardAgent yes

configж–Үд»¶еҗҢзә§зӣ®еҪ•дёӢпјҢдҪҝз”ЁдёӢйқўе‘Ҫд»ӨпјҢеҲӣе»ә tencent_keyж–Үд»¶пјҢд№ӢеҗҺжҠҠдёӢиҪҪзҡ„еҜҶй’ҘпјҲиҝҷдёӘдјҡеңЁдә‘жңҚеҠЎеҷЁдә§е•ҶеӨ„з”ҹдә§пјүеӨҚеҲ¶е…¶дёӯе№¶дҝқеӯҳгҖӮ

vim  tencent_key

иҝӣиЎҢиҝһжҺҘ

ssh tencent

иҝһжҺҘд№ӢеҗҺзҡ„з»“жһңжҳҜиҝҷж ·зҡ„

еҰӮжһңйҒҮеҲ°дәҶдёӢеӣҫиҝҷж ·зҡ„й”ҷиҜҜпјҢеҸҜд»ҘжҠҠ~/.sshж–Үд»¶еӨ№зҡ„ known_hostsпјҢknown_hosts.oldж–Үд»¶еӨ№еҲ йҷӨеҗҺеҶҚиҜ•дёҖдёӢгҖӮ

иҝҷж—¶е°ұеҸҜд»ҘжҠҠйңҖиҰҒзҡ„зҺҜеўғе®үиЈ…дёҠеҺ»дәҶгҖӮ

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install cmake
sudo apt-get install gcc
sudo apt-get install g++

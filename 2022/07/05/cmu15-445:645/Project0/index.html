本文最后更新于：2022年7月10日 凌晨

Project Specification

实现三个类: Matrix，RowMatrix和 RowMatrixOperations，这些矩阵是简单的二维矩阵，必须支持加法、矩阵乘法和简化的一般矩阵乘法(GEMM)运算。

只需修改一个文件：p0_starter.h，可以在项目的src/include/primer/p0_starter.h下找到。

Matrix 抽象类定义了 RowMatrix派生类的公共函数， RowMatrixOperations 类使用 RowMatrix 对象来完成上面提到的加法，乘法和GEMM运算。函数原型和成员变量已在文件中指定。该项目要求您填写所有构造函数、析构函数和成员函数的实现，不要添加任何其他函数原型或成员变量。

下面是仍需要安装的东西，这些最好在你进行第二步的cmake ..之前完成

  • clang-fornat：brew install clang-format

  • build tool：brew install make，Clion的Ninja会报错，需要切换为make

  • gcc/g++：brew install gcc

  • clang-tidy：

    brew install llvm ln -s "$(brew --prefix llvm)/bin/clang-tidy" "/usr/local/bin/clang-tidy"

  • valgrind：不支持mac os 11以上版本了，不知道后续是否可以继续维护。

评测账号申请：进入CMU FAQ，根据对应的操作进行申请 gradescope 账号就好。
